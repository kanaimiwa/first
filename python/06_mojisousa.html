文字列操作 len find []を使ってみよう

文字列に使える便利な命令 幾つか

  • len：文字数を調べる
  • fint：検索をする。見つからなかったときは−1を返す
  • [][start:end]：切り出しをする
  • 文字の位置は0から始まる

使用例


    s = "abcdefghi"

    #文字数を調べる
    print len(s)

    #文字の検索
    print s.find("x")

    #文字の切り出し
    print s[2] #２番目の切り出し c
    print s[2:5] #2から5未満の切り出し　bcd
    print s[:5] #始めから5未満の切り出し
    print s[2:] #2番目から最後までの切り出し
    print s[2:-1] #2番目から後ろから数えて1番目未満の切り出し bcdefgh
