リストで使える命令 sort、reverse、split、joinを使おう

命令

  • sales = [255, 100, 353, 400, 'abc' ]

リストを操作する命令

  • sort：小さい順にならべかえる
  • reverse：並び順を逆にしてくれる
  • split：文字列を、ある文字で分離させて、リスト化する
  • join：リストを、ある文字で接続して、文字列かする

使い方


    salse =[50, 100, 80, 45]
    sales.sort() #小さい順位ならべかえる
    sales.reverse() #逆順にする

    d = "2016/12/12"
    print d.split("/") #"/"で分離して、リスト化する
    a = ["a", "b", "c"]
    print "-".join(a) #"-"で接続して、文字列化する
