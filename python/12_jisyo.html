辞書を使ってみよう

辞書・・・keyとvalueが一緒になったもの

辞書の表現方法

  • sales = {"taguchi":200, "fkoji":200, "dotinstall":500}
  • printした時には、この順番通りには出ない

辞書を操作する命令

  • in：keyの存在チェック true or false
  • keys：keyの一覧
  • values：値の一覧
  • items：両方の一覧

使い方


    print sales["tagchu"] #taguchiのvalueが見たい
    sales["fkoji"] = 800 #fkojiのvalueに800を代入する
    print sales　#fkojiのvalueが変更されている

    print "taguchi" in sales
    print sales.keys()
    print sales.values()
    print sales.items()
