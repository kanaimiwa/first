リストを使ってみよう

他言語では、配列と呼ばれている

表現方法

  • sales = [255, 100, 353, 400, 'abc' ]

リストを操作する命令

  • len：数
  • +：連結
  • *：繰り返し
  • []：切り出し
  • in：存在チェック true or false
  • range：リストを一気に作る方法、単なる数字の連番

    print len[sales] #5
    print sales[2] #353
    sales[2] = 100 #salesの2に100を代入
    print sales[2] #100
    print 100 in sales #true
    print range(10) #0から10未満のリストを作ってくれる
    print range(3, 10) #3から10未満のリストを作ってくれる
    print range(3, 10, 2) #3から10未満のリストを2飛ばしで作成してくれる
