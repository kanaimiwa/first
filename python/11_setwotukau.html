セット（集合型）を使ってみよう

セット・・・重複を許さないリスト

セットの表現方法

  • a = set([1, 2, 3, 4])
  • 数字が重複していると、後ろに来るものは無視される

タプルを操作する命令

  • -：差集合
  • |：和集合
  • ＆：積集合
  • ^：どちらかにしかないもの

使い方


    a = set([1, 2, 3, 4])
    b = set([3, 4, 5])
    print a - b # aにあってbにないもの　1,2
    print a | b # 1, 2, 3, 4, 5
    print a & b # 3, 4 重複したとこだけ表示される
    print a ^ b # 1, 2, 5 どちらかにしかないもの
